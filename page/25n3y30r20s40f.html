英语笔记（五十二）

上次课中 我们已经学习了 【完成时态 】和【被动语态 】的结合， 表

达的含义是 ”某个动作已经被做 ”，它的形式可以表达为“ have been

done ”， 其中 be done 表达动作的 【被动 】，have 和 been 表达 【现在完

成】的状态，这个结构共用了 been 这个词来连接前后两个部分。

我们 先来单独看一下后面的部分， be 和 done 表达的是被动语态，

要求 be 后面添加动词的 ed 形式 。

其实我们还学习过一个类似的形式 ，是使用 be 添加动词的 ing 形

式，也就是 be doing ，所表达的含义是 【正在进行 】，那么可不可以把

完成时态和正在进行时态结合起来呢？

其实是可以的， 我们只需要把最后的 ed 动词更换为 ing 形式的动

词即可，整体表达为 have been doing 。2

相同的道理， been 也是共用的部分，也 起到了承上启下的作用，

它需要使用过去分词形式和 have 一起表达完成 ，同时 和 doing 一起表

达了正在进行 ，三个词一起构成了 现在 完成进行时态。

这听起来好像有点复杂，等一下我们多练习一下你就明白了 。
