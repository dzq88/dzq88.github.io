闲置安卓手机当代理服务器

方法：

1.先将安卓手机ROOT,参考Android刷机与root
手机最好还是刷一个安卓原生或者类原生系统
2.下载Proxy Server，并安装在旧手机上

建议：作为代理服务器使用（手机的IP地址最好设为静态IP，不然每次重启手机或者各个设备来使用代理服务器上网，都要设置一遍各设备的代理网关地址）

手机设置静态IP的方法有很多：

直接在手机上设置
在手机当前连接的WIFI设置中，将DHCP改成静态，然后输入自己需要的IP地址，（注意：IP地址要和路由器自动分配IP在同一网段）
举例：
路由器自动分配IP：192.168.0.111
自己设置的静态IP：192.168.0.222 （尾数222别超过255都能用）

DNS1设置成路由器的:192.168.0.1(根据自己实际的路由器品牌给的登录地址设置)

保存后，手机的IP地址就不会随意变动了

打开Proxy Server进行设置

点击Proxy Server顶部ADD，
Server name处随便设置一个代理服务器名称，
例如：daili

Run on port 下面设置一个端口号，
例如：65535（不能大于这个数，否则无效，也不能设为0）
滑屏返回或按返回键返回，问是否保存，选Yes即保存

点击Proxy Server顶部的info查看此代理服务器手机的IP地址，

点击Proxy Server顶部Start启动代理服务器

此时这台手机就成为了一台代理服务器

去其它设备上填写代理服务器手机的IP地址加设置的端口号，
即可通过代理服务器上网

更改网关

局域网全局设备更改：修改主路由器的DHCP设置，网关改为代理服务器手机的IP，点击保存并退出；（此种改法，以后连接主路由器的所有设备上网流量，都会走代理服务器手机）

单一设备更改：更改单台设备的网关与代理端口号，网关为代理服务器手机的IP地址，代理端口号为Proxy Server代理服务器手机里设置的端口号

保存之后，只是，改的这台设备上网流量走代理服务器手机。

电脑端更改

1.打开edge
2.点击edge浏览器顶部右上方的三个小点
3.在下拉栏里点击设置选项
4.点击系统和性能
5.点击打开计算机的代理设置
6.手动设置代理->使用代理服务器（开关打开）
7.地址栏输入：Proxy Server顶部的info查看到的代理服务器手机的IP地址
8.端口栏输入：Proxy Server代理服务器手机里设置的端口号
9.保存即可走代理服务器手机上网
