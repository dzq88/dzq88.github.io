Linux一般用法

下载

wget命令

wget是linux最常用的下载命令, 一般的使用方法是: wget + 空格 + 要下载文件的url路径

例如: # wget http://www.linuxsense.org/xxxx/xxx.tar.gz

解压

unzip命令

例如：unzip xxxx/xxx.tar.gz //解压文件到当前目录

创建目录

mkdir 命令

1.创建单级目录
mkdir A //你要创建A目录

2.创建多级目录
mkdir -p A/B //你要创建的A一层目录/B二层目录

touch 命令 创建文件

touch 你想要创建的文件和文件后缀

去到目录或文件

cd

  1. 命令格式
    cd 目录名
    2.回到根目录
    cd 或 cd / 又或者 cd ~
    说明：进入系统根目录,上面命令执行完后拿ls命令看一下

目录的移动

mv

mv A B //移动A目录到B目录

将test目录下的文件log1.txt,log2.txt,log3.txt移动到目录test3中。
mv log1.txt log2.txt log3.txt /home/路径目录/test3

or

mv -t /home/路径目录/test3/ log1.txt log2.txt log3.txt

删除目录或文件

rmdir 命令

1.rmdir A //只能删除空目录A

rm -rf 命令 强制删除

1.递归删除
rm -rf 你要删除的目录
